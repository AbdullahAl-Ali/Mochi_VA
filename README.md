# Mochi_VA
A simple Python-based voice assistant that listens, understands, and responds to basic commands. Built using beginner-friendly libraries for speech recognition and text-to-speech. Easy to customize, extend, and learn from. Perfect starter project for voice-controlled automation.

This project uses speech recognition and text-to-speech (TTS) to provide a hands-free assistant experience similar to a Google Voice Assitent.

- Greet the user according to the time of day (morning, afternoon, evening)

- Wikipedia search with spoken summary

- Open websites like Google, Facebook, YouTube

- Play random music from a specified folder

- Open system applications: Calculator, Notepad, CMD

- Open Calendar (Google Calendar via browser)

- Tell jokes and respond to basic small talk

- Exit gracefully with a voice command

## How to run? 

1. Create a virtual environment:

   ```bash
   conda create -n mochi python=3.11 -y
   ```

2. Activate virtual environment:

   ```bash
   conda activate mochi
   ```

3. Install required packages:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the MOCHI script:

   ```bash
   python mochi.py
   ```
